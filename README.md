# Event Registration Form (UI Design)

This project is a front-end event registration form created using HTML and CSS. It is designed to practice form elements and improve skills in building structured and user-friendly UI layouts.



# Features

- Clean and modern event registration form UI
- Input fields for participant details
- Event selection dropdown
- Radio buttons for preferences/options
- Checkbox for terms and conditions
- Submit button design



# Technologies Used

- HTML5
- CSS



# What I Practiced

In this project, I learned and practiced:

- Creating structured forms using <form> tag
- Using different input types (text, email, number, etc.)
- Dropdown selection using <select>
- Radio buttons for choosing options
- Checkbox for agreement/terms
- CSS styling for layout, spacing, and alignment



# Project Purpose

The purpose of this project is to simulate a real-world event registration system UI and improve understanding of:
- HTML form structure
- Input handling (frontend only)
- UI/UX design fundamentals
